pub struct ApiIdentityPolicy(_);

The API identity policy

Uses either the Session or a JWT (JSON Web Token) for request identity storage.

The constructor takes a key as an argument. This is the private key for the JWTs - if this key is changed, all Tokens, which are signed with the old key are invalid.
